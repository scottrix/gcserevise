GC3: Digital Design Tools and Techniques

Foundation Higher All Boards

Vector vs raster graphics; Adobe Photoshop, Illustrator, InDesign; layers, masks, paths; resolution and file formats; colour modes (RGB, CMYK); digital illustration; photo manipulation.

Graphic Communication: Digital Design Tools and Techniques

Vector vs raster graphics; Adobe Photoshop, Illustrator, InDesign; layers, masks, paths; resolution and file formats; colour modes (RGB, CMYK); digital illustration; photo manipulation.

Key Fact: Vector graphics (Illustrator) use mathematical paths — they scale infinitely without quality loss; ideal for logos and illustrations.
Key Fact: Raster images (Photoshop) are pixel-based — they lose quality when enlarged; ideal for photo editing and digital painting.
Key Fact: Resolution matters: 72ppi for screen, 300ppi for print; RGB colour mode for screen, CMYK for print.
Key Fact: Layers, masks and non-destructive editing allow you to experiment without permanently altering your original image.

❓ Practice Questions

Q1: Create a series of experimental studies exploring digital design tools and techniques. Annotate your process.

Q2: Research an artist or designer who works with digital design tools and techniques. Write a critical analysis of their approach and explain how it has informed your own work.

Q3: Evaluate the effectiveness of your experiments with digital design tools and techniques. Which approaches were most successful and why?

✅ Answers

  1. A range of experimental studies showing different approaches to digital design tools and techniques, each annotated with media, technique and purpose. Show development from initial experiments to refined outcomes.
  2. A critical analysis naming a specific artist/designer, using specialist vocabulary, explaining their context and approach, and making clear connections to personal work.
  3. A reflective evaluation identifying successful and less successful experiments, explaining WHY using technical and conceptual language, and showing how evaluation led to further refinement.

🎯 Exam Tips

  • Address ALL four Assessment Objectives — AO1 (develop), AO2 (refine), AO3 (record), AO4 (present)
  • Show the FULL creative journey — from initial research to final outcome
  • Annotate your work using specialist vocabulary (not just "I like it")
  • Draw from primary sources wherever possible — real objects, not photos
  • Experiment with a RANGE of media and techniques before selecting
  • Make clear connections between artist research and your own developing work

📝 Exam Technique

Graphic Communication Exam Tips:
Show SCREENSHOTS of your working process. Include layers panel, tools used, settings. Explain WHY you chose each tool. Compare digital and hand-rendered versions of the same idea.

⚠️ Common Errors

Watch Out!

Students often make mistakes with this topic. Wrong: Photoshop is the best tool for all graphic design — logos, posters, everything. Correct: Photoshop is for RASTER (pixel) work — photo editing, digital painting. Illustrator is for VECTOR work — logos, icons, illustrations that need to scale. InDesign is for LAYOUT — multi-page documents. Choose the right tool for the task.

✍️ Model Answer

Full-Mark Response

Use both vector and raster software to develop a design project. Show how you combine tools and techniques, annotate your workflow, and explain why each tool was appropriate for each task.

A grade 9 response will: demonstrate competent use of multiple software tools; explain why each tool was chosen; show a professional workflow; annotate with technical vocabulary (paths, masks, layers, vectors, resolution).

Mark scheme: AO1 (develop through investigation) + AO2 (refine through experimentation) + AO3 (record ideas and observations) + AO4 (present personal response) = total marks

Detailed Notes

Vector vs Raster Graphics

Vector graphics use mathematical paths (points, lines, curves) to define shapes. They scale infinitely without losing quality — ideal for logos, icons and typography. Created in Adobe Illustrator, Inkscape, or Affinity Designer. Raster (bitmap) graphics are grids of pixels. They have a fixed resolution and lose quality when enlarged — ideal for photographs and textured images. Edited in Adobe Photoshop, GIMP, or Affinity Photo.

Example

A company logo must work on both a business card (10mm) and a billboard (10m). A vector file (e.g. .AI or .SVG) scales perfectly to both sizes. If the logo were a raster image, enlarging it to billboard size would make it pixelated and blurry.

Industry-Standard Software

GCSE Graphic Design students should understand key software: Adobe Illustrator (vector illustration, logos, layouts), Adobe Photoshop (photo editing, raster manipulation, compositing), Adobe InDesign (multi-page layout for magazines and brochures). Free alternatives include Inkscape (vector), GIMP (raster), and Canva (simple design). Each tool has specific strengths — Illustrator for precision, Photoshop for photo-realism, InDesign for typography-heavy layouts.

Example

A magazine designer would use InDesign to create the page layout with text columns and image placeholders, Photoshop to retouch the cover photograph, and Illustrator to create vector graphics like charts or the magazine logo. All three files can be linked together in an InDesign document.

File Formats and Output

Understanding file formats is essential: .AI/.SVG for vector source files, .PSD for Photoshop source files, .PDF for print-ready output (preserves vectors and fonts), .JPEG for compressed photographs (lossy), .PNG for web graphics with transparency, .GIF for simple animations. Print requires 300dpi resolution and CMYK colour mode; screen uses 72dpi and RGB. Choosing the wrong format causes quality loss or enormous file sizes.

Example

A student submitting GCSE coursework should save source files as .PSD or .AI (preserving layers for editing) and export final designs as .PDF for print (A2 poster at 300dpi CMYK) or .PNG for digital portfolio (72dpi RGB with transparency for the background).

Extended Practice

Q: Explain why a graphic designer would use vector graphics for a logo rather than raster graphics.

Q: A designer needs to create a double-sided A5 flyer for a local event. Describe the software and file format choices they should make and justify each decision.

Extended Answers

  1. A logo must be reproduced at many sizes — from a tiny favicon (16x16 pixels) to a large shop sign (several metres). Vector graphics are defined by mathematical paths, not pixels, so they scale to any size without losing sharpness or becoming pixelated. Vector files are also much smaller than high-resolution raster files because they store path data, not millions of individual pixel values. Additionally, vector graphics allow easy colour changes — a logo can be switched from full colour to single-colour (e.g. for embossing or foil stamping) by changing fill properties. Raster logos would need to be recreated at each size, risking inconsistency and quality loss.
  2. The designer should use InDesign for the page layout because it handles text formatting, multiple pages, and precise alignment better than Illustrator or Photoshop. Vector graphics (logos, icons) should be created in Illustrator and placed into InDesign as linked files. Photographs should be edited in Photoshop (colour correction, cropping) at 300dpi CMYK for print quality. The final output should be exported as a press-ready PDF with 3mm bleed, crop marks, and all fonts embedded. This ensures the printer receives a file that will reproduce exactly as designed. A JPEG would be unsuitable because it loses quality with compression, does not support CMYK properly for print, and cannot include bleed or crop marks.
