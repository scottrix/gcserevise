Command Words & Question Types

Understanding WJEC Eduqas C690QS exam questions

Key Fact: 'State' requires a brief factual answer without explanation; just give the fact or definition directly.
Key Fact: 'Describe' requires saying what happens or what a process is, without explaining why it happens.
Key Fact: 'Explain' requires giving reasons — say what happens and why, linking cause and effect in your answer.
Key Fact: 'Compare' requires stating similarities and differences between two or more things; use comparative language.
Key Fact: 'Calculate' requires showing a numerical method: write the formula, substitute values and give the final answer with units.
Key Fact: 'Evaluate' requires making a judgement using evidence; weigh up strengths and weaknesses and reach a supported conclusion.
Key Fact: 'Suggest' requires a plausible answer based on knowledge, even if the specific context is unfamiliar.
Key Fact: 'Discuss' requires a balanced argument covering multiple viewpoints before reaching a conclusion.
Key Fact: 'Analyse' requires breaking down information to examine relationships, patterns or underlying principles.
Key Fact: Short-answer questions (1-2 marks) need concise responses; extended-answer questions (3-6 marks) need developed points with clear structure.
Key Fact: Always check the number of marks: it indicates how many separate points or steps are expected in the answer.
Key Fact: Read the entire question before starting; look for key command words and the specific context of the scenario.

📋 Key Vocabulary and Concepts

For Command Words & Question Types, you must know:

  • Command word: The instruction word in an exam question (e.g. describe, explain, calculate) that tells you what type of response is required.
  • State: Give a fact, name or value without explanation; a brief, direct answer is sufficient.
  • Explain: Give reasons for something; say what happens and why, showing the cause-and-effect relationship.
  • Compare: Identify similarities and differences between two or more items, using comparative language such as 'whereas' or 'however'.
  • Evaluate: Assess the strengths and weaknesses of something and make a supported judgement or conclusion.
  • Suggest: Propose a plausible answer using your knowledge, even if the specific context is not directly covered in the course.

❓ Practice Questions

Q: What is the difference between 'state' and 'explain' in an exam question?

Q: A question asks you to 'compare half-wave and full-wave rectification'. What should your answer include?

Q: How should you approach a 'calculate' question worth 3 marks?

Q: What does 'evaluate' require that 'describe' does not?

Q: Why is it important to check the number of marks for a question?

✅ Answers

  1. 'State' needs only a brief factual answer with no reasoning; 'explain' requires you to give reasons, linking cause and effect.
  2. Both similarities (both convert AC to DC using diodes) and differences (full-wave uses both half-cycles, half-wave only one; full-wave has higher output and less ripple).
  3. Write the formula (1 mark), substitute the correct values (1 mark), and give the final numerical answer with correct units (1 mark).
  4. 'Evaluate' requires a judgement supported by evidence, weighing up strengths and weaknesses; 'describe' only needs a factual account of what happens.
  5. The number of marks indicates how many separate points or steps are expected; a 4-mark question needs roughly four distinct points or development stages.

🎯 Exam Tips

  • Underline the command word and the key subject in each question before you start writing.
  • For 'explain' questions, use 'because', 'therefore' or 'due to' to clearly link cause and effect.
  • In 'compare' answers, use a structured format: point about A, then matching point about B, for each feature.
  • For 'calculate' questions, always show your working — marks are awarded for the method, not just the answer.
  • Match the length of your answer to the marks: 1 mark = one sentence; 4 marks = four developed points.

📝 Exam Technique

GCSE Electronics Exam Tips:
Use precise electronics terminology in all answers. When performing calculations, always show working and include units. For circuit analysis, state which law or rule you are applying (e.g. Ohm's law, Kirchhoff's laws). For design questions, justify component choices with calculations. For evaluation questions, consider performance, cost and practical constraints. WJEC Eduqas C690QS rewards answers that demonstrate understanding through use of specific electronic evidence and accurate calculations.

⚠️ Common Errors

✗ Explaining when the question only asks you to state. ✓ If the command word is 'state', give a brief factual answer only; save time by not adding unnecessary explanation.

✗ Describing when the question asks you to explain. ✓ 'Explain' requires reasons and cause-effect links; just saying what happens (describing) will not gain full marks.

✗ Listing features separately instead of comparing them side by side. ✓ For 'compare' questions, present each point as a direct comparison: 'A does X whereas B does Y.'

✗ Giving a one-word answer to a 3-mark question. ✓ Check the mark allocation; a 3-mark question needs three distinct points or steps of working.

✍️ Model Answer

Full-Mark Response

Explain the difference between the command words 'describe', 'explain' and 'evaluate', and show how each would be used in answering a question about a Zener diode voltage regulator.

'Describe' requires stating what happens without reasons: 'A Zener diode is connected in reverse bias with a series resistor. When the input voltage varies, the output voltage across the Zener remains approximately constant.' 'Explain' requires giving reasons: 'The Zener diode maintains a constant voltage because it operates in reverse breakdown, where the voltage across it changes very little for large changes in current. The series resistor absorbs the excess voltage and limits the current to protect the diode.' 'Evaluate' requires making a judgement using evidence: 'A Zener regulator is simple and low-cost, making it suitable for low-power applications. However, it has limited accuracy and poor efficiency because the series resistor dissipates power as heat. For applications requiring precise regulation or high current, a three-terminal IC regulator would be more appropriate.' Each command word requires a progressively deeper level of response: what, why, and how good.
