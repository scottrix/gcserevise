Engineering Drawings & CAD/CAM

Orthographic projection, isometric drawing, section views, assembly drawings and CAD/CAM workflows.

Key Fact: Engineering drawings communicate design intent precisely using standardised conventions and symbols.
Key Fact: Orthographic projection shows multiple 2D views: front elevation, plan and side elevation.
Key Fact: Third angle projection places views in the orientation seen from that direction (the UK standard).
Key Fact: Isometric drawing shows a 3D view at 30 degrees with no perspective distortion; all axes at true scale.
Key Fact: Section views show internal features by cutting through the component; hatching indicates cut material.
Key Fact: Assembly drawings show how components fit together, with a parts list and item numbers.
Key Fact: Dimensioning rules: dimensions must be clear, not duplicated, and placed to show functional relationships.
Key Fact: Tolerances specify acceptable variation using +/- values or limit dimensions.
Key Fact: Surface finish symbols indicate required smoothness (Ra value in micrometres).
Key Fact: CAD creates 3D models and 2D drawings using software such as SolidWorks or Fusion 360.
Key Fact: CAM converts CAD data into machine instructions (G-code) for CNC machines.
Key Fact: CAD/CAM integration: the 3D model generates toolpaths automatically, linking design to production.

📋 Key Vocabulary and Concepts

For Engineering Drawings & CAD/CAM, you must know:

  • Orthographic projection: A drawing showing multiple 2D views of a 3D object arranged in standard positions.
  • Third angle projection: The UK standard where each view is placed on the side from which it is seen.
  • Isometric drawing: A 3D representation at 30 degrees with all axes at true scale and no perspective.
  • Section view: A drawing showing internal features by cutting through; hatching indicates solid material.
  • Assembly drawing: A drawing showing how components fit together, with item numbers and a parts list.
  • G-code: The programming language controlling CNC machines, specifying tool movements and speeds.

❓ Practice Questions

Q: Explain the difference between orthographic and isometric drawings.

Q: Why is a section view used in engineering drawings?

Q: Describe the information in an assembly drawing.

Q: Explain the CAD/CAM workflow from design to manufactured part.

Q: State three dimensioning rules.

✅ Answers

  1. Orthographic shows separate 2D views (front, plan, side) for precise dimensional information. Isometric shows a single 3D view for easier visualisation but less precision for dimensioning.
  2. A section reveals internal features (holes, chambers, wall thicknesses) hidden in external views. Hatching distinguishes solid material from empty space.
  3. All components in assembled positions with item numbers, a parts list with descriptions, quantities and materials. Overall dimensions but usually not individual part dimensions.
  4. 1) Create 3D CAD model. 2) Generate 2D drawings with dimensions. 3) Use CAM to create toolpaths. 4) Post-process to G-code. 5) Verify by simulation. 6) Run CNC machine.
  5. 1) Never duplicate a dimension. 2) Place dimensions between views where possible. 3) Dimension from a common datum for related features.

🎯 Exam Tips

  • Always name the specific drawing type (orthographic, isometric, section, assembly).
  • Third angle projection is the UK standard - state this explicitly.
  • Section views: hatching identifies different materials and cut surfaces.
  • Assembly drawings show HOW parts fit; detail drawings show EACH part individually.
  • CAD/CAM: describe the full chain from 3D model to CNC output.

📝 Exam Technique

GCSE Engineering Exam Tips:
Use precise engineering terminology in all answers. When performing calculations, always show working and include units. For material selection, justify choices by linking properties to requirements. For process questions, explain why a method is suitable for the material, quantity and tolerance. For evaluation questions, consider performance, cost, manufacture and sustainability. AQA 8852 and WJEC Eduqas 5799QA reward answers that demonstrate understanding through specific technical evidence and accurate calculations.

⚠️ Common Errors

✗ Isometric drawings show true shapes on all faces. ✓ Isometric distorts circular features into ellipses and does not show true shapes on non-isometric faces.

✗ Section views are only needed for complex components. ✓ Section views are used whenever internal features need clarity, even for simple components.

✗ CAD has completely replaced hand drawing. ✓ Hand drawing remains important for quick sketches, site modifications and situations without CAD access.

✗ Assembly drawings should include all dimensions for each component. ✓ Assembly drawings show overall dimensions; individual dimensions belong on separate detail drawings.

✍️ Model Answer

Full-Mark Response

An engineer needs to communicate a bracket design with an internal bore and two mounting holes to a manufacturer. Describe the drawing package required. [6 marks]

The package requires: an orthographic detail drawing in third angle projection with front, plan and side elevation showing full dimensions and tolerances. A section view through the bore showing internal diameter, wall thickness and depth with hatching. An isometric view to help the manufacturer visualise the 3D shape. Together these provide complete, unambiguous manufacturing information.
