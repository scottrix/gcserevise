AoS4 Study Piece: Bartók Hungarian Pictures

Bartók: Hungarian Pictures — Movement 1: Evening in the Village; Movement 2: The Bear Dance; Movement 4: Slightly Tipsy; Movement 5: Swineherd's Dance. First assessed 2026. (Movement 3: Melody is NOT included.) Folk-derived material; modal/pentatonic melodies; irregular metre; orchestration; string techniques; polymodal harmony.

Key Fact: Movement 1 (Evening in the Village) alternates between slow modal folk melody and lively dance — depicts a peaceful evening scene in rural Hungary.
Key Fact: Movement 2 (The Bear Dance) uses heavy, irregular rhythm and rough string techniques to depict a clumsy bear — humour through musical characterisation.
Key Fact: Movement 4 (Slightly Tipsy) depicts drunkenness through unstable rhythm, sliding pitches and wavering dynamics — musical word painting.
Key Fact: Movement 5 (Swineherd's Dance) is fast and rhythmic with virtuosic string writing — folk dance energy translated into orchestral brilliance.

⚠️ Common Errors

Watch Out!

Students often make mistakes here. Wrong: For Hungarian Pictures you just need to describe what each movement sounds like. Correct: The exam requires CRITICAL APPRAISAL using musical vocabulary: identify specific elements (modal melody, irregular metre, polymodal harmony, string techniques), explain HOW they create the depicted scene, and analyse Bartók's transformation of folk material. Description alone is insufficient.

✍️ Model Answer

Full-Mark Response

Critically appraise two movements from Bartók's Hungarian Pictures. Analyse how Bartók uses musical elements to create character and depict the scene in each movement.

A grade 9 response will: identify folk-derived elements in each movement; explain how specific musical elements create the depicted character; analyse Bartók's orchestration and string techniques; show understanding of polymodal harmony; compare movements; use precise AQA vocabulary.

Mark scheme: AO3 (analyse and evaluate) + AO4 (knowledge and understanding) = total marks for listening; AO1 (perform) + AO2 (compose) for coursework components

📊 AO Deep Dive

Assessment Objective Analysis

GCSE Music has three components: Understanding Music (40% — written exam with listening and study-piece questions), Performing Music (30% — one solo and one ensemble performance), Composing Music (30% — one free composition and one to a brief). The four Assessment Objectives are: AO1 Perform with technical control, expression and interpretation (30%); AO2 Compose and develop musical ideas with technical control and coherence (30%); AO3 Analyse and evaluate music using musical vocabulary (20%); AO4 Demonstrate knowledge and understanding of musical elements, contexts and language (20%). For grade 9, you must show exceptional technical control, creative and coherent composition, perceptive analysis using precise terminology, and sophisticated understanding of how musical elements create meaning in context.
