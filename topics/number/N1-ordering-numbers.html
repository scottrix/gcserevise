N1: Ordering Numbers

Foundation Higher All Boards

Order positive and negative integers, decimals and fractions; use the symbols =, ≠, <, >, ≤, ≥

📋 Key Concepts

Definition: Ordering numbers means arranging them from smallest to largest (ascending) or largest to smallest (descending).

Comparison Symbols

SymbolMeaningExample
=Equal to5 = 5
≠Not equal to3 ≠ 7

Number Line

Negative numbers are smaller than positive numbers. The further left on a number line, the smaller the value.

... -3, -2, -1, 0, 1, 2, 3 ...
(left = smaller, right = larger)

📝 Ordering Integers

Rule: Negative numbers are always smaller than positive numbers. The larger the negative number (ignoring the minus sign), the smaller its value.
Example 1

Order from smallest to largest: -5, 3, -1, 0, 2

Solution: -5, -1, 0, 2, 3

Explanation: -5 is furthest left on the number line, so it is the smallest.

Example 2

Which is smaller: -7 or -3?

Solution: -7 is smaller than -3

Explanation: -7 is further left on the number line.

📝 Ordering Decimals

Method: Compare digits from left to right, starting with the highest place value. Use place value columns if needed.
Example 3

Order from smallest to largest: 0.45, 0.5, 0.398, 0.42

Solution: 0.398, 0.42, 0.45, 0.5

Tip: Add zeros to make them the same length: 0.450, 0.500, 0.398, 0.420

📝 Ordering Fractions

Method 1: Convert all fractions to have the same denominator (common denominator)
Method 2: Convert all fractions to decimals
Example 4

Order from smallest to largest: 3/4, 2/5, 1/2

Solution (using common denominator):

Common denominator = 20

3/4 = 15/20, 2/5 = 8/20, 1/2 = 10/20

Order: 2/5, 1/2, 3/4

Example 5

Order from smallest to largest: 2/3, 5/8, 7/10

Solution (using decimals):

2/3 ≈ 0.667, 5/8 = 0.625, 7/10 = 0.7

Order: 5/8, 2/3, 7/10

📝 Mixed Numbers

Tip: Convert mixed numbers to improper fractions or decimals for easier comparison.
Example 6

Order from smallest to largest: 2 1/4, 2.5, 2 3/8

Solution:

2 1/4 = 2.25, 2.5 = 2.5, 2 3/8 = 2.375

Order: 2 1/4, 2 3/8, 2.5

❓ Practice Questions

Q1: Order from smallest to largest: -8, 4, -2, 0, 6

Q2: Order from smallest to largest: 0.7, 0.08, 0.71, 0.8

Q3: Order from smallest to largest: 1/2, 2/5, 3/10

Q4: Which symbol makes this correct? -5 ___ -2

Q5: Order from smallest to largest: 1.5, 1 1/3, 1.25, 1 3/4

✅ Answers

  1. -8, -2, 0, 4, 6
  2. 0.08, 0.7, 0.71, 0.8
  3. 3/10, 2/5, 1/2 (same as: 0.3, 0.4, 0.5)
  4. -5 < -2 (negative 5 is smaller)
  5. 1.25, 1 1/3, 1.5, 1 3/4

🎯 Exam Tips

  • Always check if you need ascending (smallest to largest) or descending (largest to smallest)
  • For decimals, add zeros to make all numbers the same length
  • For fractions, convert to decimals if the denominators are hard to compare
  • Draw a number line for negative numbers if you get confused
  • Remember: -10 is smaller than -1
