N7: Integer Indices

Foundation Higher All Boards

Calculate with integer indices; negative and fractional indices (Higher)

рҹ“Ӣ Key Concepts

Index Laws: Rules for manipulating expressions with powers. These apply when the base numbers are the same.
The Six Index Laws:
1. Multiply: aбөҗ Г— aвҒҝ = aбөҗвҒәвҒҝ
2. Divide: aбөҗ Г· aвҒҝ = aбөҗвҒ»вҒҝ
3. Power of power: (aбөҗ)вҒҝ = aбөҗвҒҝ
4. Power of zero: aвҒ° = 1 (a вү  0)
5. Negative indices: aвҒ»вҒҝ = 1/aвҒҝ
6. Fractional indices: a^(1/n) = вҒҝвҲҡa

рҹ“қ Multiplication Rule

Rule: When multiplying with the same base, ADD the powers: aбөҗ Г— aвҒҝ = aбөҗвҒәвҒҝ
Example 1

Simplify: 3вҒҙ Г— 3ВІ

Solution: 3вҒҙ Г— 3ВІ = 3вҒҙвҒәВІ = 3вҒ¶

Check: 81 Г— 9 = 729 = 3вҒ¶ вң“

Example 2

Simplify: xвҒө Г— xвҒ·

Solution: xвҒө Г— xвҒ· = xвҒөвҒәвҒ· = xВ№ВІ

рҹ“қ Division Rule

Rule: When dividing with the same base, SUBTRACT the powers: aбөҗ Г· aвҒҝ = aбөҗвҒ»вҒҝ
Example 3

Simplify: 5вҒ· Г· 5Ві

Solution: 5вҒ· Г· 5Ві = 5вҒ·вҒ»Ві = 5вҒҙ

Check: 78125 Г· 125 = 625 = 5вҒҙ вң“

рҹ“қ Power of a Power

Rule: When raising a power to another power, MULTIPLY them: (aбөҗ)вҒҝ = aбөҗвҒҝ
Example 4

Simplify: (2Ві)вҒҙ

Solution: (2Ві)вҒҙ = 2ВіЛЈвҒҙ = 2В№ВІ

Check: 8вҒҙ = 4096 = 2В№ВІ вң“

рҹ“қ Negative Indices (Higher)

Rule: A negative index means the reciprocal: aвҒ»вҒҝ = 1/aвҒҝ
Example 5

Evaluate: 4вҒ»ВІ

Solution: 4вҒ»ВІ = 1/4ВІ = 1/16

Example 6

Evaluate: 2вҒ»Ві Г— 2вҒө

Solution: 2вҒ»ВівҒәвҒө = 2ВІ = 4

Example 7

Write 1/8 as a power of 2

Solution: 1/8 = 1/2Ві = 2вҒ»Ві

рҹ“қ Fractional Indices (Higher)

Rules:
a^(1/2) = вҲҡa
a^(1/3) = ВівҲҡa
a^(m/n) = (вҒҝвҲҡa)бөҗ or вҒҝвҲҡ(aбөҗ)
Example 8

Evaluate: 9^(1/2)

Solution: 9^(1/2) = вҲҡ9 = 3

Example 9

Evaluate: 8^(2/3)

Solution: 8^(2/3) = (ВівҲҡ8)ВІ = 2ВІ = 4

Or: 8^(2/3) = ВівҲҡ(8ВІ) = ВівҲҡ64 = 4

Example 10

Evaluate: 16^(вҲ’3/2)

Solution: 16^(вҲ’3/2) = 1/(16^(3/2)) = 1/(вҲҡ16)Ві = 1/4Ві = 1/64

вқ“ Practice Questions

Q1: Simplify: aвҒҙ Г— aвҒ¶

Q2: Simplify: (xВі)вҒө

Q3: Simplify: 7вҒё Г· 7Ві

Q4 (Higher): Evaluate: 5вҒ»ВІ

Q5 (Higher): Evaluate: 27^(2/3)

вң… Answers

  1. aВ№вҒ°
  2. xВ№вҒө
  3. 7вҒө
  4. 5вҒ»ВІ = 1/25
  5. 27^(2/3) = (ВівҲҡ27)ВІ = 3ВІ = 9

рҹҺҜ Exam Tips

  • Only use index laws when bases are the same
  • Negative indices don't make numbers negative - they make fractions
  • For a^(m/n): denominator = root, numerator = power
  • Practice converting between fractional indices and roots
  • Check answers by expanding: e.g., 2Ві Г— 2ВІ = 8 Г— 4 = 32 = 2вҒө вң“
