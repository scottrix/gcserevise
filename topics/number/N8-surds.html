N8: Surds

Higher Only All Boards

Calculate exactly with fractions, multiples of ПҖ; simplify surds, rationalise denominators

рҹ“Ӣ Key Concepts

What is a Surd? A surd is an irrational square root that cannot be written as an exact decimal. Examples: вҲҡ2, вҲҡ3, вҲҡ5, вҲҡ6...
Important: вҲҡ4 = 2 is NOT a surd (it's rational). вҲҡ2 вүҲ 1.414... is a surd (it's irrational and cannot be written exactly).
Key Surd Rules:
вҲҡ(a Г— b) = вҲҡa Г— вҲҡb
вҲҡ(a Г· b) = вҲҡa Г· вҲҡb
(вҲҡa)ВІ = a
вҲҡaВІ = a (for a вүҘ 0)

рҹ“қ Simplifying Surds

Method: Find the largest square number that divides into the surd. Split it and simplify.
Example 1

Simplify: вҲҡ12

Solution:

вҲҡ12 = вҲҡ(4 Г— 3) = вҲҡ4 Г— вҲҡ3 = 2вҲҡ3

Example 2

Simplify: вҲҡ50

Solution:

вҲҡ50 = вҲҡ(25 Г— 2) = вҲҡ25 Г— вҲҡ2 = 5вҲҡ2

Example 3

Simplify: вҲҡ72

Solution:

вҲҡ72 = вҲҡ(36 Г— 2) = вҲҡ36 Г— вҲҡ2 = 6вҲҡ2

Alternative: вҲҡ72 = вҲҡ(9 Г— 8) = 3вҲҡ8 = 3 Г— 2вҲҡ2 = 6вҲҡ2

рҹ“қ Adding and Subtracting Surds

Rule: Only "like surds" can be added or subtracted. Simplify first, then combine like terms.
Example 4

Simplify: вҲҡ12 + вҲҡ27

Solution:

вҲҡ12 = 2вҲҡ3 and вҲҡ27 = 3вҲҡ3

2вҲҡ3 + 3вҲҡ3 = 5вҲҡ3

Example 5

Simplify: 3вҲҡ20 - вҲҡ45

Solution:

вҲҡ20 = 2вҲҡ5, so 3вҲҡ20 = 6вҲҡ5

вҲҡ45 = 3вҲҡ5

6вҲҡ5 - 3вҲҡ5 = 3вҲҡ5

рҹ“қ Multiplying Surds

Rule: Multiply surds by multiplying the numbers inside: вҲҡa Г— вҲҡb = вҲҡ(a Г— b)
Example 6

Simplify: вҲҡ2 Г— вҲҡ8

Solution:

вҲҡ2 Г— вҲҡ8 = вҲҡ16 = 4

Example 7

Expand: вҲҡ3(2 + вҲҡ12)

Solution:

вҲҡ3 Г— 2 = 2вҲҡ3

вҲҡ3 Г— вҲҡ12 = вҲҡ36 = 6

Answer: 6 + 2вҲҡ3

рҹ“қ Rationalising Denominators

Why? A fraction should not have a surd in the denominator. Multiply top and bottom to remove it.

Simple Denominators

Method: If the denominator is вҲҡa, multiply top and bottom by вҲҡa.
Example 8

Rationalise: 1/вҲҡ2

Solution:

Multiply by вҲҡ2/вҲҡ2:

1/вҲҡ2 Г— вҲҡ2/вҲҡ2 = вҲҡ2/2

Example 9

Rationalise: 6/вҲҡ3

Solution:

6/вҲҡ3 Г— вҲҡ3/вҲҡ3 = 6вҲҡ3/3 = 2вҲҡ3

Complex Denominators

Method: If the denominator is a Вұ вҲҡb, multiply by the "conjugate" a вҲ“ вҲҡb.
Example 10

Rationalise: 1/(2 + вҲҡ3)

Solution:

Multiply by (2 - вҲҡ3)/(2 - вҲҡ3):

1(2 - вҲҡ3) / [(2 + вҲҡ3)(2 - вҲҡ3)]

= (2 - вҲҡ3) / (4 - 3)

= (2 - вҲҡ3) / 1 = 2 - вҲҡ3

Example 11

Rationalise: 5/(3 - вҲҡ2)

Solution:

Multiply by (3 + вҲҡ2)/(3 + вҲҡ2):

5(3 + вҲҡ2) / [(3 - вҲҡ2)(3 + вҲҡ2)]

= 5(3 + вҲҡ2) / (9 - 2)

= 5(3 + вҲҡ2) / 7 = (15 + 5вҲҡ2)/7

рҹ“қ Exact Values with ПҖ

Exact values: Leave answers in terms of ПҖ when asked for exact values.
Example 12

Find the exact area of a circle with radius 5 cm

Solution:

A = ПҖrВІ = ПҖ Г— 5ВІ = 25ПҖ cmВІ

Do NOT calculate 25 Г— 3.14... - leave as 25ПҖ

вқ“ Practice Questions

Q1: Simplify: вҲҡ75

Q2: Simplify: вҲҡ18 - вҲҡ8

Q3: Simplify: 2вҲҡ5 Г— 3вҲҡ10

Q4: Rationalise: 3/вҲҡ5

Q5: Rationalise: 2/(1 + вҲҡ3)

вң… Answers

  1. вҲҡ75 = вҲҡ(25 Г— 3) = 5вҲҡ3
  2. вҲҡ18 = 3вҲҡ2, вҲҡ8 = 2вҲҡ2, so 3вҲҡ2 - 2вҲҡ2 = вҲҡ2
  3. 2 Г— 3 Г— вҲҡ50 = 6 Г— 5вҲҡ2 = 30вҲҡ2
  4. 3/вҲҡ5 Г— вҲҡ5/вҲҡ5 = 3вҲҡ5/5
  5. 2(1 - вҲҡ3)/(1 - 3) = 2(1 - вҲҡ3)/(-2) = вҲҡ3 - 1

рҹҺҜ Exam Tips

  • Always simplify surds fully - look for the largest square factor
  • When adding/subtracting, check all surds are simplified first
  • Memorise: (a + вҲҡb)(a - вҲҡb) = aВІ - b
  • Leave answers as surds unless asked for a decimal
  • "Exact value" means leave in terms of ПҖ or as a surd

рҹ§  Problem-Solving Strategies

Problem-Solving

For surd problems, always simplify first before combining. When rationalising, identify whether the denominator is a simple surd (multiply by вҲҡa) or a bracketed expression (multiply by the conjugate). For exact value problems, never convert to decimals вҖ” leave answers in surd form or in terms of ПҖ.
Multi-Step Problem

A rectangle has length 2 + вҲҡ3 and width 2 вҲ’ вҲҡ3. Show that the area is an integer, and find the perimeter in simplified surd form.

Solution: Area = (2 + вҲҡ3)(2 вҲ’ вҲҡ3) = 4 вҲ’ 3 = 1 (integer вң“). Perimeter = 2(2 + вҲҡ3) + 2(2 вҲ’ вҲҡ3) = 4 + 2вҲҡ3 + 4 вҲ’ 2вҲҡ3 = 8.

вҡ пёҸ Common Errors

Watch Out!

1. Wrong: вҲҡ12 + вҲҡ3 = вҲҡ15 (adding inside one square root) Correct: вҲҡ12 = 2вҲҡ3, so вҲҡ12 + вҲҡ3 = 2вҲҡ3 + вҲҡ3 = 3вҲҡ3

2. Wrong: вҲҡ(a + b) = вҲҡa + вҲҡb Correct: вҲҡ(a + b) вү  вҲҡa + вҲҡb вҖ” you cannot split a square root over addition

3. Wrong: Rationalising 1/(3 + вҲҡ2) by multiplying by вҲҡ2/вҲҡ2 Correct: Multiply by the conjugate (3 вҲ’ вҲҡ2)/(3 вҲ’ вҲҡ2) to eliminate the surd from the denominator

вңҚпёҸ 6-Mark Exam Question

Extended Answer

6 marks: (a) Simplify вҲҡ28 вҲ’ вҲҡ63 + вҲҡ175. (b) Rationalise the denominator of 4/(3 вҲ’ вҲҡ5). (c) Show that (2 + вҲҡ5)ВІ = 9 + 4вҲҡ5. Hence find the exact value of (2 + вҲҡ5)ВІ + (2 вҲ’ вҲҡ5)ВІ.

(a) вҲҡ28 = 2вҲҡ7, вҲҡ63 = 3вҲҡ7, вҲҡ175 = 5вҲҡ7. So 2вҲҡ7 вҲ’ 3вҲҡ7 + 5вҲҡ7 = 4вҲҡ7.

(b) 4/(3 вҲ’ вҲҡ5) Г— (3 + вҲҡ5)/(3 + вҲҡ5) = 4(3 + вҲҡ5)/(9 вҲ’ 5) = 4(3 + вҲҡ5)/4 = 3 + вҲҡ5.

(c) (2 + вҲҡ5)ВІ = 4 + 4вҲҡ5 + 5 = 9 + 4вҲҡ5 вң“. By symmetry, (2 вҲ’ вҲҡ5)ВІ = 9 вҲ’ 4вҲҡ5. Sum = 9 + 4вҲҡ5 + 9 вҲ’ 4вҲҡ5 = 18.

Mark scheme: 2 marks for (a) with full simplification, 2 marks for (b) with correct conjugate method, 2 marks for (c) including the sum

рҹ“Ҡ AO3: Reason & Interpret

Reasoning and Interpretation

A right-angled triangle has legs of length вҲҡ8 cm and вҲҡ18 cm.

(a) Show that the hypotenuse has exact length вҲҡ26 cm.

(b) A student says "The hypotenuse is вҲҡ8 + вҲҡ18 = 2вҲҡ2 + 3вҲҡ2 = 5вҲҡ2 cm." Explain why this is wrong.

(c) Compare the exact hypotenuse with the student's answer. Which is longer?

Answers: (a) By Pythagoras: hypВІ = вҲҡ8ВІ + вҲҡ18ВІ = 8 + 18 = 26, so hyp = вҲҡ26 cm. (b) The student added the legs, but Pythagoras' theorem requires SQUARING the legs first, then adding, then square rooting. You cannot just add the sides. (c) вҲҡ26 вүҲ 5.10, 5вҲҡ2 вүҲ 7.07. The student's answer is longer, which shows adding sides gives an overestimate of the hypotenuse.
