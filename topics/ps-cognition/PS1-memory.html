Memory

How memory works: encoding, storage and retrieval. The multi-store model, types of long-term memory, and factors affecting the accuracy of memory including eyewitness testimony.

Key Fact: Memory is the process of encoding, storing and retrieving information. Encoding converts sensory input into a storable form; storage retains it; retrieval accesses it when needed.
Key Fact: The Multi-Store Model (Atkinson & Shiffrin, 1968): Sensory memory (brief, capacity limited by senses) -> Short-term memory (limited capacity 7+/-2 items, duration ~30 seconds) -> Long-term memory (potentially unlimited capacity and duration) via rehearsal.
Key Fact: Types of LTM: Episodic (personal experiences, e.g. your first day at school), Semantic (facts and knowledge, e.g. Paris is the capital of France), Procedural (skills and actions, e.g. riding a bike).
Key Fact: Eyewitness testimony is unreliable: Loftus & Palmer (1974) showed leading questions distort recall - participants estimated faster speeds when verbs like 'smashed' were used. Anxiety, weapon focus and post-event discussion also reduce accuracy.
Key Fact: Improving memory: elaborative rehearsal (linking new info to existing knowledge), chunking (grouping items), and mnemonic devices enhance encoding and retrieval.

❓ Practice Questions

Q1: Describe the multi-store model of memory and evaluate its strengths and limitations.

Q2: Explain how leading questions can affect the accuracy of eyewitness testimony, referring to Loftus & Palmer's research.

Q3: Compare and contrast episodic, semantic and procedural long-term memory.

✅ Answers

  1. The MSM provides a clear, testable framework: evidence supports the distinction between STM and LTM (case of HM, primacy-recency effect). Limitations: it oversimplifies memory - STM and LTM are not unitary stores (WMM shows STM has multiple components); rehearsal is not the only way to transfer to LTM (flashbulb memories).
  2. Loftus & Palmer showed verb manipulation ('smashed' vs 'hit') changed speed estimates by up to 10 mph. This demonstrates that leading questions alter memory through response bias or actual memory distortion. However, real-world eyewitness testimony may be more accurate than lab studies due to higher anxiety and real consequences.
  3. Episodic: personal events, time-stamped, consciously recalled. Semantic: general facts, not time-stamped, consciously recalled. Procedural: skills/habits, not consciously recalled. All three are distinct LTM types with different brain regions.

🎯 Exam Tips

  • STM has a capacity of 7+/-2 items and a duration of about 30 seconds.
  • Eyewitness testimony is unreliable - always mention Loftus & Palmer when discussing EWT.
  • Episodic = events you experienced. Semantic = facts you know. Procedural = things you can do.

📝 Exam Technique

Psychology Exam Tips:
When evaluating memory research, use the GRAVE framework: Generalisability, Reliability, Applications, Validity, Ethics. Always compare lab studies (high control, low ecological validity) with real-world studies (high ecological validity, less control).

⚠️ Common Errors

Watch Out!

Students often make mistakes here. Wrong: The multi-store model fully explains how memory works because it shows information flows from sensory to short-term to long-term memory. Correct: While the MSM describes a basic flow of information, it fails to account for: working memory (Baddeley & Hitch showed STM has multiple components), different types of LTM (episodic, semantic, procedural), and the fact that some memories form without conscious rehearsal (e.g. flashbulb memories). The model is a useful starting point but is oversimplified.

✍️ Model Answer

Full-Mark Response

Evaluate the usefulness of the multi-store model of memory for understanding how we remember and forget.

A grade 9 response will: outline the MSM with evidence (case of HM supports STM/LTM distinction, primacy-recency effect); evaluate strengths (clear framework, testable predictions, supported by brain scanning); evaluate limitations (oversimplifies STM and LTM, ignores working memory, doesn't explain procedural memory); conclude: the MSM was foundational and is supported by key evidence, but it is now considered an incomplete explanation - the Working Memory Model and LTM subtypes offer more comprehensive accounts of specific memory processes.
