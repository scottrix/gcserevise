R6: Comparing Texts

Foundation Higher All Boards AO3

Comparing texts and identifying similarities and differences in perspectives and methods.

📋 What is Text Comparison?

Definition: Comparison involves identifying similarities and differences between texts, considering perspectives, methods, and how writers convey their viewpoints.

Comparison involves:

  • Identifying each writer's perspective or viewpoint
  • Comparing how these perspectives are similar or different
  • Analysing the methods each writer uses
  • Using evidence from both texts
Key Focus:
Compare PERSPECTIVES + Compare METHODS

📝 What to Compare

1. Perspectives and Viewpoints

  • What is each writer's stance on the topic?
  • Are their opinions similar or different?
  • How strongly does each writer feel?
  • Is there bias or objectivity?
Example

Source A: Argues that social media has a positive impact on young people's communication.

Source B: Claims social media isolates young people and harms real-world relationships.

These perspectives directly oppose each other.

2. Methods

  • What language devices does each writer use?
  • How do they structure their arguments?
  • What tone does each writer adopt?
  • How do they appeal to the reader?
Comparing Methods

While Source A uses statistics and expert opinion to build credibility, Source B relies on emotive language and personal anecdote to create sympathy.

3. Tone and Style

Tone Effect
Formal Authoritative, serious, credible
Informal Relatable, conversational, accessible
Emotive Passionate, persuasive, moving
Objective Balanced, factual, neutral

🔍 How to Structure a Comparison

Method 1: Integrated Approach

Write about both texts in each paragraph, using comparative connectives to link them.
Example Integrated Paragraph

Both writers express strong opinions about climate change, but their perspectives differ significantly. Source A presents a scientific perspective, arguing that "immediate action is essential," while Source B adopts a more sceptical stance, suggesting that "the evidence remains inconclusive." While Source A uses authoritative language like "undeniable" and "urgent," Source B employs rhetorical questions like "Can we really be certain?" to create doubt.

Method 2: Point-by-Point

  • Make a point about Text A
  • Compare with Text B
  • Use comparative language
  • Repeat for each comparison point

Comparative Connectives

Similarity Difference
Similarly However
Likewise In contrast
Both Whereas
Equally On the other hand
In the same way Unlike

⚠️ Common Mistakes to Avoid

Mistake Correction
Writing about texts separately Use integrated approach - compare throughout
Describing content without comparing Focus on similarities and differences
Only comparing content, not methods Compare both perspectives AND methods
Using only one text for evidence Quote from BOTH texts equally
Not using comparative language Use connectives: similarly, however, whereas

📊 Mark Scheme Guidance

AQA Paper 2 Question 4 (16 marks)

  • Level 4 (13-16 marks): Perceptive comparison with detailed analysis
  • Level 3 (9-12 marks): Clear comparison with clear analysis
  • Level 2 (6-8 marks): Some comparison with some comment
  • Level 1 (3-5 marks): Simple comparison with limited support
Examiner's Tip: You must compare BOTH perspectives AND methods. Use evidence from both texts equally. Spend about 20-25 minutes on this question.

❓ Practice Questions

Q1: What are the two main things you should compare between texts?

Q2: What is the difference between an integrated approach and writing about texts separately?

Q3: Name three comparative connectives for showing differences.

Q4: Why should you use evidence from both texts?

✅ Answers

  1. Perspectives (viewpoints) and methods (how they convey those viewpoints).
  2. Integrated approach compares within each paragraph using connectives; separate approach writes about each text individually without comparing.
  3. However, in contrast, whereas, on the other hand, unlike.
  4. To show you can analyse both texts and make valid comparisons - it proves your comparison is based on evidence.

🎯 Exam Tips

  • Read both texts and identify each writer's viewpoint
  • Use comparative connectives throughout
  • Write about both texts in each paragraph
  • Compare methods as well as perspectives
  • Quote from both texts equally
  • Look for similarities AND differences
  • Don't analyse language in depth - focus on comparison
