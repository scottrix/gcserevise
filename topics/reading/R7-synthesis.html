R7: Synthesis

Foundation Higher All Boards AO1 AO3

Combining information from multiple sources to create a coherent summary.

📋 What is Synthesis?

Definition: Synthesis means combining information from different sources to create a summary. You draw together ideas from both texts to show common themes or differences.

Synthesis involves:

  • Reading and understanding both texts
  • Identifying relevant information from each
  • Combining information into a coherent whole
  • Using evidence from both sources
Key Difference:
Synthesis = What the texts SAY (content)
Comparison = How the texts CONVEY it (methods)

📝 Types of Synthesis Questions

1. Summary Questions (AQA Paper 2 Q2)

Example Question

Write a summary of the differences between the two writers' experiences.

You should use details from both sources.

2. "What Do You Learn" Questions

Example Question

Using details from both sources, write a summary of what you learn about...

Key Points About Synthesis

  • You're summarising CONTENT, not analysing methods
  • Use your own words where possible
  • Include short, relevant quotations
  • Focus on what each text SAYS
  • Don't analyse language or structure

🔍 How to Write a Synthesis

Step-by-Step Approach

Step 1: Read both texts and identify the key information
Step 2: Decide what each text says about the topic
Step 3: Find similarities and/or differences
Step 4: Write in your own words, using brief quotations
Step 5: Organise logically - group related points

Synthesis Sentence Starters

Purpose Starters
Introducing similarities Both texts suggest... / Similarly, both sources... / The texts agree that...
Introducing differences While Source A... Source B... / In contrast... / The texts differ in that...
Combining information Both sources indicate... / Together, these texts reveal... / The evidence from both suggests...
Example Synthesis

Both texts describe experiences of childhood, but they present very different perspectives. Source A describes a carefree, happy childhood where the writer "ran wild through the fields" and felt "completely free." In contrast, Source B presents a more restricted experience, where the writer felt "trapped" by strict rules and describes a childhood of "constant supervision." While Source A's writer recalls "endless summers," Source B remembers feeling "watched constantly" and unable to escape adult scrutiny.

📊 Synthesis vs Comparison

Aspect Synthesis Comparison
Focus WHAT texts say HOW texts convey meaning
Skills Summary, combining Analysis of methods
Language analysis No - focus on content Yes - analyse techniques
Quotations Brief, for evidence For analysis
Own words Yes - preferred Embedded quotes
Remember: In synthesis questions, don't analyse language devices. Focus on summarising what each text says about the topic.

⚠️ Common Mistakes to Avoid

Mistake Correction
Analysing language devices Focus on content - what each text says
Writing about only one text Use evidence from BOTH sources
Copying large chunks of text Summarise in your own words
Not addressing the question focus Check you're answering what's asked
Listing points without synthesis Combine information from both texts

📊 Mark Scheme Guidance

AQA Paper 2 Question 2 (8 marks)

  • Level 4 (7-8 marks): Perceptive synthesis with relevant detail
  • Level 3 (5-6 marks): Clear synthesis with clear detail
  • Level 2 (3-4 marks): Some synthesis with some detail
  • Level 1 (1-2 marks): Simple information with limited detail
Examiner's Tip: This is NOT a language analysis question. Focus on what each text SAYS - the content and ideas. Use your own words and keep quotations brief. Spend about 10 minutes on this question.

❓ Practice Questions

Q1: What is the main difference between synthesis and comparison?

Q2: Should you analyse language devices in a synthesis question?

Q3: How should you use quotations in synthesis?

Q4: What sentence starter would you use to show a difference between texts?

✅ Answers

  1. Synthesis focuses on WHAT texts say (content); comparison focuses on HOW they convey meaning (methods).
  2. No - synthesis is about summarising content, not analysing language.
  3. Keep them brief and use them as evidence to support your summary points.
  4. "While Source A... Source B..." or "In contrast..." or "Unlike Source A, Source B..."

🎯 Exam Tips

  • Focus on content, not language analysis
  • Use evidence from both texts equally
  • Write in your own words where possible
  • Use synthesis sentence starters
  • Don't write too much - 8 marks needs a concise response
  • Group related points logically
  • Answer the specific question focus
