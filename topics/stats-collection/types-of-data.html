Types of Data

Understand how to classify data as qualitative or quantitative, discrete or continuous, categorical or ordinal, and recognise bivariate data.

Key Fact: Qualitative data is non-numerical — it describes qualities or categories (e.g. eye colour, favourite subject).
Key Fact: Quantitative data is numerical — it can be measured or counted (e.g. height, number of pets).
Key Fact: Quantitative data splits into discrete and continuous.
Key Fact: Discrete data can only take specific values, usually whole numbers (e.g. number of siblings, shoe size).
Key Fact: Continuous data can take any value within a range, including decimals (e.g. height, time, temperature).
Key Fact: Categorical data is a type of qualitative data where values fall into named groups with no order (e.g. blood group, gender).
Key Fact: Ordinal data has categories that can be ordered or ranked (e.g. satisfaction level: poor, OK, good, excellent).
Key Fact: Grouped data is continuous data organised into class intervals (e.g. 0 ≤ x < 10).
Key Fact: Bivariate data involves two variables measured for each individual — used to investigate relationships (e.g. height and weight).
Key Fact: Primary data is collected by the investigator; secondary data comes from existing sources.
Key Fact: Knowing the data type determines which diagram and which statistic you should use.
Key Fact: Continuous data is displayed using histograms; discrete data can use bar charts; ordinal data can use ordered bar charts.

📋 Key Vocabulary and Concepts

For Types of Data, you must know:

  • Qualitative data: Non-numerical data that describes categories or qualities.
  • Quantitative data: Numerical data that can be measured or counted.
  • Discrete data: Quantitative data that can only take specific (usually whole-number) values.
  • Continuous data: Quantitative data that can take any value within a range, including decimals.
  • Ordinal data: Qualitative data where the categories have a natural order or ranking.
  • Bivariate data: Data that involves two variables measured for each individual, used to explore relationships.

❓ Practice Questions

Q: Is 'number of goals scored in a match' discrete or continuous?

Q: Give an example of ordinal data.

Q: What is bivariate data?

Q: Why is 'height' classified as continuous data?

Q: What type of diagram is most appropriate for displaying continuous grouped data?

✅ Answers

  1. Discrete — it can only take whole-number values (0, 1, 2, …).
  2. Satisfaction rating (very dissatisfied, dissatisfied, neutral, satisfied, very satisfied).
  3. Data that consists of pairs of values for two variables measured on the same individual, used to investigate relationships.
  4. Because height can take any value within a range, including decimal values like 165.3 cm.
  5. A histogram.

🎯 Exam Tips

  • If a value is counted, it is usually discrete; if it is measured, it is usually continuous.
  • Shoe size is discrete even though half-sizes exist — it can only take set values.
  • Temperature is continuous even though it is recorded to a set number of decimal places.
  • When asked to classify data, give your answer and a brief justification.
  • Bivariate data questions often link to scatter graphs and correlation.

📝 Exam Technique

GCSE Statistics Exam Tips:
Use precise statistical terminology in all answers. When performing calculations, always show working including the formula, substitution and final answer. For interpretation questions, always refer back to the context and use data to support your conclusions. For evaluation questions, consider reliability, validity, bias and alternative explanations. Edexcel 1ST0 and AQA 8382 reward answers that demonstrate understanding through correct use of statistical notation and evidence-based reasoning.

⚠️ Common Errors

✗ Classifying shoe size as continuous because it has half sizes ✓ Shoe size is discrete — it can only take specific values (6, 6.5, 7, 7.5, …), not any value in between.

✗ Saying ordinal data is quantitative because it uses numbers like 1st, 2nd, 3rd ✓ Ordinal data is qualitative — the numbers or ranks represent ordered categories, not measurable quantities.

✗ Confusing categorical and ordinal data ✓ Categorical data has no natural order (e.g. eye colour); ordinal data has a ranking (e.g. small, medium, large).

✗ Thinking continuous data cannot be grouped ✓ Continuous data is often grouped into class intervals for display in histograms or frequency tables.

✍️ Model Answer

Full-Mark Response

Classify each of the following as qualitative (categorical or ordinal), discrete quantitative or continuous quantitative: (a) hair colour, (b) exam grade (A–U), (c) time taken to run 100 m, (d) number of books read last month.

(a) Hair colour — qualitative categorical: it is non-numerical and the categories (blonde, brown, black, red, etc.) have no natural order. (b) Exam grade (A–U) — qualitative ordinal: the grades are non-numerical categories but they do have a natural ranking from A (best) to U (worst). (c) Time taken to run 100 m — continuous quantitative: time can take any value within a range, including decimals (e.g. 12.37 seconds). (d) Number of books read last month — discrete quantitative: it is numerical but can only take whole-number values (0, 1, 2, 3, …).
